SQLSQL简介SQL (Structured Query Language:结构化查询语言) 是用于管理关系数据库管理系统（RDBMS） SQL 对大小写不敏感：SELECT 与 select 是相同的。 三大范式1.第一范式(1NF):确保每列具有原子性，即每个单元格中的数据都是不可再分的。表中的每个字段都应该是原子的，不应该包含重复的数据 2.第二范式(2NF):要求满足第一范式，并且消除非主键列对于主键的部分依赖。每个非主键列必须完全依赖于主键，而不是依赖于主键的部分属性。 3.第三范式(3NF):要求满足第三范式，并且消除主键列之间的传递依赖。任何非主键列不能依赖其他非主键列。 4.巴斯-科德范式(BCNF):要求满足第三范式，并且消除主键依赖非主键的情况。任何主键都不能依赖非主键。 范式的级别越高，数据重复性越低，更新操作时的数据一致性和完整性也更容易维护。但过度范式化可能导致查询复杂度增加，因此在实际设计数据库时需要权衡范式和实际需求之间的关系。 需要注意的是，范式化只是数据库设计的一部分，具体的设计还需要考虑性能优化、数据访问模式等因素，以及根据具体业务需求进行适 ...

回溯

定义

回溯法，是一种搜索的方式。拥有递归就会有回溯。

特点

回溯法不高效，类似于穷举。使用暴力法解决动态for。故某些题型使用回溯时，会导致运算超时。

可将回溯法抽象为树。

  1. 组合
  2. 切割
  3. 子集
  4. 排列
  5. 棋盘问题

动态规划

定义

Dynamic Programming，简称DP

状态推导

求一个问题的最优解，且该问题能分解成若干个子问题，子问题也可分解

特点

  1. 求最优解
  2. 最优解依赖各个子问题的最优解
  3. 子问题还有重叠的更小问题
  4. 从上往下分析问题，从下往上求解问题

流程

  1. 确定dp数组以及下标含有
  2. 确认递推公式
  3. 初始化dp数组
  4. 确认遍历顺序
  5. 打印数组

解决问题

  1. 斐波那契数列
  2. 爬楼梯
  3. 背包问题
  4. 剪绳子

KMP算法

定义

Knuth-Morris-Pratt字符串查找算法

特点

  1. 前缀: 包含首字母，不包含尾字母的所有子串

  2. 后缀: 包含尾字母，不包含首字母的所有子串

  3. 前缀表: 前后缀相等的值，最长相等前后缀

  5. 时间复杂度O(n+m)

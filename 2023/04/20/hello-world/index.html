快速上手hexo

官方文档中文版:hexoDocs

官方文档:hexo.io

安装前提

在此，仅提供windows下的安装，其他系统请查看官方文档。后续的配置仅供参考

git

windows: git

node.js

windows: Node.js

安装Hexo

至此，已经将所需软件全部安装完成

初始化配置以及上传至github

hexo建站

  1. 选择一个文件夹，调用以下命令，初始化hexo

    $ hexo init

  2. 运行服务器

    $ hexo server

此时本机可通过访问地址 http://localhost:4000/ 网页
3. 生成静态网页

`` 
$ hexo generate
``

git配置

如果你已经完成了git基本配置，如个人信息user.name、user.email、SSH key的生成，则可跳过该项

  1. 配置个人信息

  2. 生成SSH key
    此操作可选择跳过，使用HTTPS来替换SSH进行自动话，注意：HTTPS请保证网络通畅，各方面的，很有可能上传是失败

上次至github

  1. 新建一个仓库，仓库名严格规范，命名为 “yourname.github.io”

  2. 此时仓库为空仓库，修改hexo的配置文件_config.yml

    为

repo可使用SSH或者HTTPS的方式
3. 安装hexo自动话部署工具


4. 上传至github

5. 访问网页
在浏览器中输入网址 http://yourname.github.io
，如果访问成功，说明配置完成
